# SeedSigner Workshop 101

A beginner-friendly presentation on building your own air-gapped Bitcoin hardware wallet using [SeedSigner](https://github.com/SeedSigner/seedsigner) — open-source, stateless, and self-sovereign.

## Topics Covered

- Why hardware wallets exist
- Risks of proprietary wallets (supply chain, data breaches, physical coercion)
- The FOSS advantage — verifiable, community-driven, no company to subpoena
- Building a SeedSigner for ~$50 (Pi Zero 2 W, display, camera, case)
- Air-gapped signing workflow with Nunchuk
